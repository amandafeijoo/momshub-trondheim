# MomsHub Trondheim

MomsHub Trondheim is a community platform concept for international mothers living in Trondheim.  
The goal is to help moms connect, discover local meetups, share support, and exchange baby items in a simple and trusted way.

## Tech Stack

- React
- TypeScript
- Vite
- Styled Components
- Framer Motion
- Three.js / React Three Fiber

## Current Features

- Premium animated landing page
- Global Three.js motion background
- Floating editorial labels
- Responsive layout
- Community, Events, Marketplace and Support sections

## Future Features

- User accounts
- Community posts
- Events and meetups
- Marketplace for baby items
- Profiles for moms
- Messaging system
- Admin moderation

## Project Status

Currently in MVP development.
